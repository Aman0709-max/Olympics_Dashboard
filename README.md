# Olympics_Dashboard

1. project objective(business problem)

“The objective of this project is to analyze historical Olympic data to identify trends in country performance, athlete participation, and sport-wise medal distribution.
From a business perspective, the goal was to provide insights that could help stakeholders like sports committees understand performance patterns and make data-driven decisions.

2. kpi & metrics(KPI indicator is a measurable value that shows how well a business or process is performing)


I defined key performance indicators to measure overall performance:”

Total Medals (Gold, Silver, Bronze)

Number of Participating Countries

Number of Sports

Medal Count by Country

Medal Count by Athlete

Gender Participation Ratio

Year-wise Growth in Participation & Medals

“These KPIs helped summarize large datasets into actionable insights.”

3.data sources and transformation

The dataset consisted of multiple tables including athlete details, country data, events, and medal records.”

Data Cleaning (Power Query):

Removed null and duplicate values

Standardized country and sport names

Corrected data types

Data Transformation:

Merged and joined tables

Created calculated columns where required

Built a star schema for better performance

“This ensured the data was accurate and optimized for analysis.

4. dax and advanced features


I used DAX to create dynamic calculations and enhance interactivity:”

Measures:

Total Medals = SUM of all medal types

Country-wise medal counts

Year-wise trends

Advanced Features:

Time intelligence (trend over years)

Filters and slicers for dynamic analysis

Drill-through for detailed insights

Conditional formatting for better readability

“This allowed users to explore the data interactively and gain deeper insights.



5. visualization and story telling

The dashboard provides a comprehensive overview of Olympic performance.”

KPI Cards
Total Medals (~39K)
Countries Participated (136)
Sports Played (66)


2. Country-wise Medal Table(matrix)

“This visual highlights which countries are leading in medal counts.”

“Users can quickly identify top-performing countries and compare performance across nations.”

3. Tree map (Sports Distribution)

“The tree map shows which sports contribute the most medals.”

“This helps identify high-impact sports and where most achievements are concentrated.”

4. Pie Chart (Gender Distribution)

“This visual shows the proportion of male vs female participation.”

“It helps analyze gender balance and participation trends.”

5. Bar Chart (Top Athletes)

“This chart highlights the top medal-winning athletes.”

“It answers who the most successful players are.”

6. Line Chart (Trend Over Time)

“This shows how participation and medal counts change over the years.”

“It helps identify growth trends and patterns over time.”


7.“I added slicers and filters so users can interact with the dashboard — for example, selecting a specific country or year dynamically updates all visuals.”

“This allows users to explore insights based on their specific needs.”


6. Business impact

This dashboard provides actionable insights such as:”
Identifying top-performing countries and athletes
Understanding which sports contribute most to medals
Analyzing gender participation trends
Tracking historical performance patterns
“These insights can help decision-makers improve training focus, allocate resources efficiently, and promote balanced participation.
